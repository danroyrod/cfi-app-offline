## ğŸ”§ **TROUBLESHOOTING**

### **Audio Not Playing?**

1. **Check browser support** - Works best in Chrome, Edge, Safari
2. **Reload the page** - Simple refresh often fixes it
3. **Check system volume** - Ensure not muted
4. **Try different browser** - Some browsers work better

### **Audio Quality Issues?**

1. **Voice sounds robotic** - Normal for Web Speech API
2. **Want better quality?** - Premium TTS coming soon
3. **Adjust speed** - Sometimes 0.75x sounds more natural

### **Playback Stops?**

1. **Check autoplay settings** - Browser may block autoplay
2. **Keep page open** - Don't switch tabs initially
3. **Enable background audio** - In browser settings

### **Progress Not Saving?**

1. **Check localStorage** - Must be enabled
2. **Same browser** - Use same browser/device
3. **Clear cache** - If issues persist, clear and restart
