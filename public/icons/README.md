# App Icons

This directory contains the PWA app icons.

## Required Icons

- `icon-192x192.png` - 192x192 pixels (for Android home screen)
- `icon-512x512.png` - 512x512 pixels (for splash screen and install prompt)

## Icon Design Guidelines

- Use the CFI/ACS theme colors (blue #1e40af)
- Include airplane or flight instructor iconography
- Ensure icons are clear and recognizable at small sizes
- Use transparent background or solid color background matching theme
